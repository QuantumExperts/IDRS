# IDRS — International Dispute Resolution Services

Static website for IDRS, providing dispute resolution services across the Asia-Pacific region.

## Deployment

Push to GitHub, then connect the repository to Netlify for automatic deployment.

## Structure

- `index.html` — Home page
- `css/style.css` — Site-wide stylesheet
- `pages/` — Service and information pages
  - `about.html` — About IDRS & Neil Kirkpatrick
  - `expert-determination.html` — Expert Determination services
  - `arbitration.html` — Commercial & Expedited Arbitration
  - `mediation.html` — Commercial Mediation
  - `dispute-boards.html` — Dispute Board services
  - `conciliation.html` — Conciliation services
  - `contact.html` — Contact form and details
